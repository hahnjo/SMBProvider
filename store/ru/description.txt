ПРОШУ УЧЕСТЬ: Разработка приложения ещё не завершена, на данный момент оно позволяет только читать файлы, нежели изменять их или создавать новые. Пожалуйста, не ставьте низкие оценки только из-за этого.
Сейчас я работаю над этим. Вы можете отследить прогресс на Github (см. ниже).

Это приложение использует новую среду обращения к хранилищам, представленную в Android 4.4 KitKat для доступа к файлам, распространяемым через SMB (aka Windows Shares).
Любое приложение сможет открыть подобные файлы посредством системного диалога. В скором времени, Вы сможете создавать новые файлы или изменять их напрямую.

Исходный код приложения распростаняется под лицензией GPLv3 и расположен на https://github.com/hahnjo/SMBProvider. Смело копируйте, изменяйте или переводите его на другие языки. Не забывайте присылать ваши изменения!
